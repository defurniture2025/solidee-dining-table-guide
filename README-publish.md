# SOLIDEÉ external SEO landing page publishing package

This folder is a standalone SEO landing page intended for a separate website/domain that sends qualified referral traffic to the main Solideé website.

## Recommended strategy

Do NOT make this page an immediate redirect. If the separate URL instantly redirects, Google usually will not rank the landing-page content itself, and the page becomes only a pass-through URL.

Better setup:

1. Publish this landing page on the separate website and allow it to be indexed.
2. Keep the page useful and complete, with original content.
3. Use strong CTAs that send visitors to the main Solideé website and WhatsApp.
4. Track outbound clicks with UTM parameters.
5. Only use a 301 redirect later if you retire this landing page or move it permanently.

## Important before publishing

1. Replace the placeholder canonical URL `https://solidee-furniture-guides.netlify.app/` in `index.html` with the final live URL on the separate website, not the Solideé main site.
2. Replace the JSON-LD WebPage `url` placeholder with the same final live URL.
3. Upload all three files to the same folder on the host.
4. Confirm the page loads and the logo image appears.
5. Test all outbound links to Solideé and WhatsApp.
6. Submit the final landing-page URL to Google Search Console if that separate website has access.
